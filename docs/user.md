# Manual de usuario

## Telegram

- `/start`: bienvenida
- `/crear_blog Blog de turismo en Putumayo`
- `/mis_blogs`
- `/ver_blog 1`
- `/editar_blog 1 | Cambia el color principal a verde`
- `/publicar 1`
- `/exportar_zip 1`
- `/eliminar_blog 1`

## Panel web

1. Inicia sesion con las credenciales del `.env`.
2. Genera un blog desde el dashboard.
3. Entra al detalle para editarlo conversacionalmente.
4. Publica o exporta el resultado.
